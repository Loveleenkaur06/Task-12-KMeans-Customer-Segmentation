# Task-12-KMeans-Customer-Segmentation

**Overview**

This project applies KMeans clustering to segment mall customers based on Annual Income and Spending Score for better business insights.

**Tools**

Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn

**Workflow**

Loaded and cleaned dataset

Dropped CustomerID

Applied feature scaling

Used Elbow Method to find optimal clusters

Trained KMeans model

Visualized and labeled customer segments

**Result**

Optimal clusters: 6

Customers grouped into meaningful spending and income segments

Segmented dataset saved as CSV

**Outcome**

Successfully performed unsupervised customer segmentation using KMeans for business analysis.
